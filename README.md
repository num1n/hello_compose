# Hello Compose

Навчальний Android-додаток на Jetpack Compose.  
Приклад створений для лабораторної роботи №1.

## Як запустити

1. Встановіть Android Studio (останню версію).
2. Встановіть SDK (мінімум API 26).
3. Клонуйте репозиторій:

```bash
git clone https://github.com/num1n/hello_compose.git
cd hello-compose
